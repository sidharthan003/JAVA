# DSA in Java

A collection of **Data Structures and Algorithms (DSA)** implementations in **Java**.  
This repository is meant for learning, practicing, and revising core computer science concepts.

---

## 📌 Contents

The repository includes implementations of:

### 1. Arrays & Strings
- Array operations (insertion, deletion, searching, sorting)
- String manipulation (palindrome, reverse, substring, etc.)

### 2. Linked Lists
- Singly Linked List
- Doubly Linked List
- Circular Linked List

### 3. Stacks & Queues
- Stack (array & linked list)
- Queue (array, linked list, circular)
- Priority Queue
- Deque

### 6. Sorting & Searching
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Binary Search & Linear Search
